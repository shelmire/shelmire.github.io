Portfolio

WebAnn - This is a web-based tool for linguistic annotation (on text, audio, and video) with a great deal of flexibility in terms of the projects it can be used for. WebAnn also has workflow management tools, workforce management tools, training tools, communication tools, and an editor that allows non-programmers to create their own annotation tools. It was built with Ruby on Rails, makes heavy use of Javascript, was recently switched to the Bourbon.io suite from Bootstrap, and interacts with several different databases from unrelated projects. This is Alex's primary project at the LDC. Unfortunately, the github is private at the moment, but hopefully this will change soon. Some source code may be available upon request.
